Dans l’optique de réalisation d’un projet en équipe, nous sommes amenés à développer une application web. Pour ce, nous devrons d’abord définir nos objectifs c’est-à-dire les fonctionnalités que devra apporter notre application tout en tenant compte des deadlines qui nous sont données. Ensuite nous choisirons un design en faisant le maquettage de notre application et enfin nous devrons choisir les technologies web à utiliser pour mettre sur pied notre application. Dans la suite, notre mission consistera à expliquer la sélection les technologies web ainsi que de l’architecture d’application web pour ce projet.
